# AI Projects

A collection of AI and machine learning projects implemented in Python. Each project is in its own folder.

## Projects

- **bayesian** – Bayesian algorithm examples.
- **image_classifier_cnn** – CNN for image classification.
- **image_classifier_transformer** – Transformer-based image classifier.
- **linear** – Linear regression examples.
- **svm** – Support Vector Machine examples.
- **tictactoe** – AI for Tic-Tac-Toe.
- **trigfunction** – Trigonometric function examples.
- **xor** – XOR problem solved with neural networks.

## Usage

1. Make sure `uv` is installed.
2. Navigate into the project folder:

   ```bash
   cd <project_folder>
   uv run main.py
